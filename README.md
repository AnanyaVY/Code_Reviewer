## AI-Powered Code Review Assistant

## Project Overview
### AI-Powered Code Review Assistant is a production-ready web application that delivers instant, comprehensive code reviews for Python and JavaScript. It combines industry-standard static analysis tools (Pylint, Bandit, ESLint) with Hugging Face CodeT5 AI model to provide actionable feedback on:

#### 1.Code Quality (style, readability, maintainability)

#### 2.Security Vulnerabilities (injection risks, shell commands)

#### 3.Performance Issues & Bug Detection

#### 4.Refactoring Suggestions

## What It Does

#### 1.Paste your code (Python or JavaScript)

#### 2.AI + Linters analyze in parallel (~30-60s)

#### 3.Get instant feedback on:

#### 4.Code quality & style violations

#### 5.Security vulnerabilities (HIGH/MEDIUM/LOW)

Logical bugs & refactoring suggestions

Readability improvements
